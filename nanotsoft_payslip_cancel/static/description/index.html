Cancel Employee Payslip & Payslip Batches

Key Features:

  • Allow users to cancel the payslip and payslip batches even if payslip status is done or paid.
  • Users who have Cancel Payslip access they are only able to cancel.
  • When payslip cancel it will also cancel and delete the related Accounting Entry even after posted.

Overview

  • From the Users Menu, there is access rights called "Cancel Payslip". Users who have access rights can cancel employee payslip & payslip batches.
  • Users will be able to see the cancel button in payslip Done or Paid status.
  • When the user cancel the payslip it will change status to rejected and related accounting entry will delete like below screenshot.
  • In Payslip Batches form view, there is button for cancel all payslip related to particular batches.
  • User can see the cancel button even if payslip batches status are done or paid.
  • When the user cancels the payslip batch, also it will cancel the related payslips of that particular batch.
  • In the payslip list view, inside the top action menu if users have cancel payslip access then only they will be able to see the Cancel option.
